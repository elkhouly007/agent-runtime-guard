# OpenClaw Integration Template

Use this template as a controlled starting point. Do not treat it as a blind installer.

## Included

- `README.md`
- `openclaw-modules.example.md`

## Template Rules

- Show exact target path before applying.
- Keep external and risky modules disabled by default.
- Preserve prompt rejection and high-risk escalation rules.
- Review tool wiring before enabling browser, notifications, MCP, or external agent flows.
