# Claude Code Integration Template

Use this template as a controlled starting point. Do not treat it as a blind installer.

## Included

- `README.md`
- `claude-modules.example.md`

## Template Rules

- Show exact target path before applying.
- Keep external and risky modules disabled by default.
- Preserve standing approval policy from `claude/AGENTS.md`.
- Review any hook, plugin, or external tool wiring before enabling it.
