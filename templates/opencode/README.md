# OpenCode Integration Template

Use this template as a controlled starting point. Do not treat it as a blind installer.

## Included

- `README.md`
- `opencode-modules.example.jsonc`

## Template Rules

- Show exact target path before applying.
- Keep MCP, plugins, and risky modules disabled by default.
- Map tool permissions to the standing approval policy.
- Review any shell, external, or global config change before enabling it.
