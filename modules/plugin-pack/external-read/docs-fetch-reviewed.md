# docs-fetch-reviewed

## Class

external-read

## Purpose

Fetch documentation or reference content from known destinations with reviewed non-sensitive payloads.

## Allowed Use

- documentation reads;
- reference retrieval from known sources;
- non-sensitive external read flows.

## Approval Boundary

Ask before sending personal or confidential data, or if the destination or payload is unclear.

## Rejection Cases

Reject use if the plugin hides the destination, hides the payload, or installs code implicitly.
