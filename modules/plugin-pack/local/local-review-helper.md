# local-review-helper

## Class

local-only

## Purpose

Support local review workflows with visible output and no hidden side effects.

## Allowed Use

- local review reminders;
- local diff-focused assistance;
- local workflow nudges.

## Approval Boundary

Ask before the plugin introduces external routing, deletion, elevated use, or global mutation.

## Rejection Cases

Reject use if the plugin hides destinations, suppresses output, or rewrites behavior to bypass approval boundaries.
