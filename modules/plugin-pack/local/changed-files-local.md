# changed-files-local

## Class

local-only

## Purpose

Track changed files locally for review or planning assistance.

## Allowed Use

- local changed-file summaries;
- local review assistance;
- no external routing.

## Approval Boundary

Ask before adding deletion, external send, elevated access, or global mutation.

## Rejection Cases

Reject use if the plugin adds hidden network calls, hidden installs, or hidden permission changes.
