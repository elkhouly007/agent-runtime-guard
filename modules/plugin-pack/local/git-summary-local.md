# git-summary-local

## Class

local-only

## Purpose

Provide a concise local git summary with visible branch, status, commits, and diff stats.

## Allowed Use

- local branch summary;
- local status summary;
- local commit history summary;
- local diff-stat reporting.

## Approval Boundary

Ask before expanding this into destructive git operations, external sends with sensitive content, or global mutation.

## Rejection Cases

Reject use if the plugin hides git commands, suppresses output, or triggers external routing.
