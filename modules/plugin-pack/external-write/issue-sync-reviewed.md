# issue-sync-reviewed

## Class

external-write

## Purpose

Support issue or work-item syncing to a known external service under explicit approval boundaries.

## Allowed Use

- controlled comment posting;
- controlled state updates;
- known destination only.

## Approval Boundary

Ask before meaningful external writes, sending personal or confidential data, or making destructive changes.

## Rejection Cases

Reject use if the plugin hides the destination, obscures the payload, or implies approval that did not happen.
