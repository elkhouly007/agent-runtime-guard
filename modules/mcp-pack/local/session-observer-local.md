# session-observer-local

## Class

local

## Purpose

Provide reviewed local session or activity observation without external routing.

## Allowed Use

- local session inspection;
- local activity summaries;
- local state awareness for orchestration.

## Approval Boundary

Ask before deletion, elevated access, or any external send.

## Rejection Cases

Reject use if the module persists hidden telemetry or routes session data outside the machine.
