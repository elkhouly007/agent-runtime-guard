# git-local

## Class

local

## Purpose

Reviewed local git assistance.

## Allowed Use

- status and diff inspection;
- branch and history inspection;
- local non-destructive git help.

## Approval Boundary

Ask before destructive history rewriting, deletion, sensitive overwrite, or external push with personal or confidential data.

## Rejection Cases

Reject use if the module hides external routing or bypasses review for push-related actions.
