# filesystem-local

## Class

local

## Purpose

Reviewed local filesystem assistance.

## Allowed Use

- local file inspection;
- local project navigation;
- local non-destructive file operations.

## Approval Boundary

Ask before deletion, destructive overwrite, elevated access, or any action outside the standing approval policy.

## Rejection Cases

Reject use if the module introduces hidden downloads, hidden external paths, or unclear write behavior.
