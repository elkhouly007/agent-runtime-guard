# form-submit-gated

## Class

write-gated

## Purpose

Represent reviewed browser submission or write-like actions that require explicit approval.

## Allowed Use

- controlled form submission;
- known destination only;
- only after user approval when required by policy.

## Approval Boundary

Ask before any use involving personal or confidential data, uploads, purchases, settings changes, or deletion.

## Rejection Cases

Reject use if the destination, form contents, or consequences are unclear.
