# site-read-reviewed

## Class

read-only

## Purpose

Perform reviewed site inspection with clear destination and no write-like action.

## Allowed Use

- page reading;
- navigation with no writes;
- non-sensitive inspection.

## Approval Boundary

Ask before the task becomes a submission, upload, settings change, purchase, or deletion.

## Rejection Cases

Reject use if the destination is hidden or the task is framed as read-only while actually causing writes.
