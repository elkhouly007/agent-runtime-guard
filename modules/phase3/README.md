# Phase 3 Modules

Phase 3 adds policy scaffolding for:

- installers;
- wrappers;
- long-lived helpers and daemons;
- integration templates.

This folder documents how to wire capabilities into real tools while preserving the standing approval policy.

Read `../../references/phase1-policy.md`, `../../references/phase2-policy.md`, and `../../references/phase3-policy.md` before extending these modules.
