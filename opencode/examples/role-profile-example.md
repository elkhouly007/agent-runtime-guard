# OpenCode Role Profile Example

## Builder

Primary local coding role. Use for scoped file changes and implementation work.

## Planner

Read-only planning role. Use before broad edits, migrations, or refactors.

## Reviewer

Read-only bug-first reviewer. Use for correctness, regressions, and missing tests.

## Security Reviewer

Read-only security role. Use for secrets, auth, input handling, file writes, and outbound data risks.

## Build Fixer

Minimal-diff repair role for local build, type, lint, or test failures.
