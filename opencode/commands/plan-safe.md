# Plan Safe

Create a detailed implementation plan using Agent Runtime Guard policy.

## Output

- requirements restatement;
- implementation phases;
- dependencies;
- risks;
- complexity estimate.

## Safe-Plus Adjustments

- planning is local-safe by default;
- do not require user confirmation for ordinary local, non-destructive execution;
- clearly flag only high-risk actions for approval;
- do not introduce external tools or networked steps by default.
