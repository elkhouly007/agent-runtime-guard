# Orchestrate Safe

Coordinate multiple roles or agents for a complex task using Agent Runtime Guard approval rules.

## Execution Style

- break the task into visible subtasks;
- assign a clear role for each subtask;
- prefer local roles first;
- use trusted external agents only after payload review;
- synthesize results into one output.

## Safe-Plus Adjustments

- do not hide destinations or payloads;
- do not imply approval that did not happen;
- flag only real high-risk steps for user approval.
