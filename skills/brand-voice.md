# Skill: Brand Voice

## Trigger

Use when establishing a writing style profile from existing content (blog posts, tweets, emails, documentation), or when ensuring that new content matches an established brand voice. Also use when the user wants to write in the style of a specific person, company, or publication.

## Process

### Phase 1 — Extract the Voice from Samples

Input: 5-10 content samples from the source (posts, articles, messages).

Analyze across these dimensions:

| Dimension | What to look for |
|---|---|
| **Sentence length** | Short and punchy vs. long and complex? |
| **Formality** | Professional/formal, conversational, casual? |
| **Person** | First-person ("I do"), we ("We build"), or third? |
| **Vocabulary level** | Simple everyday words vs. technical/specialized? |
| **Tone** | Authoritative, warm, direct, humorous, earnest? |
| **Structure** | Lists and headers vs. flowing prose? |
| **Punctuation patterns** | Em dashes? Ellipses? Short paragraphs? |
| **Signature phrases** | Recurring words, metaphors, openers, closers? |
| **What they avoid** | Jargon? Passive voice? Hedge words? |

### Phase 2 — Produce the Voice Profile

```markdown
## Voice Profile: [Name/Brand]

### Character
[2-3 sentences describing the "personality" of the writing]

### Tone
[Direct, warm, authoritative, playful, etc.] — never [passive, hedging, formal, etc.]

### Sentence Style
[Short, declarative / Long, structured / Mix with pattern: ...]

### Vocabulary
[Simple and accessible / Domain-specific / Technical precision without jargon]

### Signature Patterns
- [Pattern 1, e.g., "Starts paragraphs with a strong claim, then supports"]
- [Pattern 2, e.g., "Uses em-dashes for mid-sentence pivots"]
- [Pattern 3, e.g., "Ends with an open question or forward-looking statement"]

### Avoid
- [Thing 1 that would break the voice]
- [Thing 2]

### Sample Transformation
Original: [neutral/generic version]
In-voice: [transformed to match the profile]
```

### Phase 3 — Apply the Profile

When writing new content in the voice:
1. Read the profile before writing.
2. Draft naturally, then review against the profile.
3. Check: sentence length matches, tone is consistent, signature patterns present.
4. Do a "voice break" check — highlight any sentences that feel off-brand.

## Output Format

- Voice profile document (Phase 2 format above).
- 3 example rewrites of neutral sentences demonstrating the voice.
- A short "do/don't" cheat sheet for quick reference.

## Constraints

- Do not invent characteristics not present in the source material — derive only from provided samples.
- More samples = more accurate profile — 5 samples minimum, 10+ for a reliable profile.
- Note when samples are inconsistent — a brand that writes differently across contexts has multiple voices, not one.
- Applying someone's voice requires their permission when producing public content — note this for user awareness.
